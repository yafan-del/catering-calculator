# 更新日志

## v1.1.0

### 新增功能
- **窗口吸附**：支持吸附到"闲鱼管家"窗口边缘，目标窗口移动时自动跟随
  - 吸附位置可配置（左 / 右 / 上 / 下）
  - 支持自定义间距
  - 支持 Windows + macOS 双平台
- **阶梯折扣**：支持按原价区间配置不同收款比例
  - 如 1-100 元收款 69%，101-200 元收款 70%
  - 输入原价时自动匹配对应区间并显示

### 优化
- 图标全部替换为 Element Plus Icons，不再使用 Emoji
- 修复 Tauri v2 环境下 `isTauri` 检测失败的问题（置顶等功能不生效）
- 移除 updater 签名验证，适配 OSS 直传更新包的发布流程
